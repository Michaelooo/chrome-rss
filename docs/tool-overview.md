# RSS Reader Chrome Extension 简介

## 能力概览

- **订阅管理**
  - 支持手动添加 RSS/Atom 源，自动解析站点图标与历史文章。
  - 侧边栏可拖拽排序；实时展示每个源的未读数量。
  - 支持 OPML 导入/导出，方便批量迁移订阅。

- **文章阅读**
  - 列表页提供快速筛选（全部 / 未读 / 收藏）与全文搜索。
  - 阅读视图支持图片、视频、代码块、公式等内容渲染，点击图片可放大。
  - 阅读时可单篇或批量标记已读/未读，并即时刷新未读统计与徽章。
  - 星标收藏实时同步到“我的收藏”分类与弹窗中的“最近未读”列表。

- **后台能力**
  - Service Worker 定时刷新订阅源，支持手动“一键刷新”。
  - 浏览器工具栏图标显示未读总数；弹窗展示最近 10 条未读文章，可直接打开。

## 安装与构建

1. **克隆仓库并安装依赖**
   ```bash
   git clone <repository-url>
   cd chrome-rss
   npm install
   ```

2. **开发模式**
   - 运行 `npm run dev` 启动 Vite 开发服务。
   - 修改代码后 Vite 会自动热更新，需在浏览器扩展页刷新加载的 `dist` 目录。

3. **生产构建**
   ```bash
   npm run build
   ```
   该命令会输出 `dist/` 目录，包含前台页面与后台脚本。

4. **在 Chrome 中加载扩展**
   - 打开浏览器 `chrome://extensions/`，开启“开发者模式”。
   - 点击“加载已解压的扩展程序…”，选择项目下的 `dist/` 目录。
   - 扩展加载后即可在工具栏看到图标。点击图标弹出窗口，或访问 `index.html` 进入阅读器主界面。

> 提示：在 `chrome://extensions/` 页面点击扩展卡片中的“刷新”可重新加载最新构建；点击“检查视图”下的 `popup.html` 链接可调试弹窗。
