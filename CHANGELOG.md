# 更新日志

本文档记录 Chrome RSS Reader 扩展的所有重要变更。

## [1.0.1] - 2026-02-12

### 新增

- 新增 CHANGELOG 文档，便于追踪版本更新

### 优化

- 主界面布局与侧边栏组件的交互体验优化
- 工具栏和弹窗界面的稳定性改进
- 发布打包脚本优化

### 修复

- 修复若干界面显示问题

---

## [1.0.0] - 2026-02-12

### 首次发布

- 支持 RSS 2.0 和 Atom 1.0 格式订阅
- 三栏布局：Feed 列表、文章列表、文章阅读器
- 支持文件夹管理，可拖拽排序
- 导入/导出 OPML 订阅
- 本地 IndexedDB 存储，保护隐私
- 深色模式支持
- 文章搜索、翻译、收藏功能
- 后台定时更新 Feed
- 未读数量徽章提醒
